Sobre o Projeto

  • Desenvolvido por Samuel Vitor

  • Desafio proposto pela Brisanet para os jovens aprendizes

  • Software de gerenciamente de projetos onde é possível conectar:

    • Membros a uma equipe
    • Projeto a uma equipe
    • Tarefas a um membro e a um projeto

  • Desenvolvido com Angular, Postgres e Go.

Informações Importantes Importante ler antes de usar este software

  • É necessário ter uma equipe para que sejam criados membros e projetos;

  • Para criar uma Tarefa deve existir tanto o projeto da tarefa, quanto o membro que realizará a tarefa.
